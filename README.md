# Ethan Pineda Blog

Hey, I'm Ethan, a first generation and low-income student (FGLI) junior at Northwestern University studying Computer Science with minors in Latino/Latina Studies and Data Scieince. My interest include full-stack software engineerning, web development, human-computer interaction, cloud computng, and machine learning. 


This blog is a place and medium for me to share my thoughts, experiences, and projects. I hope to use this blog as a way to document my journey as a student, software engineer, and as a person. I hope that this blog can be a resource for others who are interested in similar topics and experiences or are generally, just curious about how I think and what I do.
